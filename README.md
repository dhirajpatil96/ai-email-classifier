## Dataset
The dataset used in this project is too large for GitHub.
Download link for dataset:
https://www.kaggle.com/datasets/wcukierski/enron-email-dataset 

## Functions of scripts.
1. loadingset1.py: To load raw email data from the original dataset and prepare it for preprocessing.
2. datacleaning2.py: To clean and preprocess raw email text so it can be used for machine learning.
3. labelling3.py: To automatically assign category and urgency labels to emails.
4. trainig4.py: To train and evaluate models for email category classification.
5. urgency_prediction5.py: To train and evaluate models for email urgency prediction.
6. urgency_inference6.py: To predict urgency of new or unseen emails in real time.
7. dashboard7.py: To provide a visual dashboard that displays email classification and urgency prediction results in an easy to understand format.
8. email_stream_simulator8.py: To simulate a real time incoming email system.
9. sample_labeled_emails.py: To create a smaller version of the labeled dataset for testing and deployment.
